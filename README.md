# Netflix Movies & TV Shows – EDA(EXPLORATORY DATA ANALYSIS)

## Overview
Performed exploratory data analysis on Netflix dataset using Pandas,Matplotlib and Seaborn.

## Key Insights
- Majority content type is Movies
- USA is the top content producer
- Most popular genres are Drama and International Movies
- Significant growth in content after 2016
- TV-MA is the dominant rating

## Tools Used
Python, Pandas,Matplotlib and Seaborn
